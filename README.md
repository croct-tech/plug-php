## Introduction

Croct is a headless CMS that helps you manage content, run AB tests, and personalize experiences without the hassle of complex integrations.

## Installation

Run this command to install the SDK:

```sh
composer require croct/plug-php
```

See our [quick start guide](https://docs.croct.com/reference/sdk/php/integration) for more details.
